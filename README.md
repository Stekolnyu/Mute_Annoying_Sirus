# muted
Mute anoying WoW sounds.

## 2.4.3
Place the `Sound` directory inside `WoW\Data\{language}\`.

## 3.3.5
Place the `Sound` directory inside `WoW\Data\{language}\`.
В нашем случае "WoW\Data\ruRU\" 

## 4.3.4
Place the `Sound` directory inside `WoW\Data\`.

## 5.4.8+
Place the `Sound` directory inside `WoW\`.
